# **App Name**: BlogSphere

## Core Features:

- User Authentication: JWT authentication with login, registration and role based access (admin/user).
- Blog CRUD: Create, read, update, and delete blog posts with role-based access control. Admin can manage all posts, regular users can only manage their own.
- Admin Dashboard: Dedicated admin interface to manage blog posts and user roles.
- Protected Routes: Implement protected routes for admin pages, accessible only by admin users.
- User Profiles: Allow users to manage and customize their public-facing profiles. 
- Database Integration: Persistent storage of blog posts and user data.

## Style Guidelines:

- Primary color: Deep violet (#673AB7) for a sense of sophistication and modernity.
- Background color: Light gray (#F0F0F0), very desaturated from the violet primary color.
- Accent color: Teal (#009688) to provide a contrasting highlight.
- Body and headline font: 'Inter' sans-serif, to give a modern neutral look.
- Minimalist icons for navigation and actions, following a consistent line style.
- Clean and responsive layout with clear content hierarchy. Use of cards to display blog previews.
- Subtle transitions and animations on interactions like hover effects on cards or loading states.
